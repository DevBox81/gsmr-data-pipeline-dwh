## Objectifs du Projet

### Objectif Global

> Mettre en place un **pipeline de données automatisé** et un **data warehouse** destiné au suivi et à l'optimisation du réseau GSM-R.

### Objectifs Détaillés

| # | Objectif | Description |
|---|----------|-------------|
| 1 | **Collecte automatisée** | Collecter automatiquement les données depuis Expandium, SharePoint et fichiers locaux |
| 2 | **Pipeline ETL/ELT** | Développer un pipeline de données fiable, documenté et monitoré |
| 3 | **Data Warehouse** | Construire un DWH structuré selon une architecture moderne (Medallion) |
| 4 | **Qualité des données** | Assurer la qualité, la cohérence et la normalisation des données |
| 5 | **Automatisation** | Automatiser la mise à jour quotidienne/hebdomadaire avec historisation |
| 6 | **KPIs centralisés** | Centraliser et standardiser les KPIs réseau |
| 7 | **Base pour le reporting** | Fournir une base solide pour l'automatisation du reporting |
| 8 | **Documentation** | Documenter l'architecture, les flux et les règles de transformation |

# Project Specifications

- **Data Ingestion**: Automatically collect data from Expandium, SharePoint, and local files.

- **ETL/ELT Pipeline**: Process data through extraction, transformation, and loading into a centralized system.

- **Data Warehouse**: Store structured data using fact and dimension tables for efficient analysis.

- **Data Quality**: Ensure consistency, cleaning, and standardization of all incoming data.

- **Automation**: Schedule regular data updates (daily or weekly) without manual intervention.

- **KPI Management**: Calculate and centralize key GSM-R network performance indicators.
